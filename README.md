# Notes-app

### [Live](https://notesapp-v1mo.onrender.com/)

## Technologies:

-   Vue 3
-   SCSS
-   Node.js
-   Express
-   MongoDB

## Setup

To run this project, install it locally using npm:

#### Server

```
$ npm install
$ npm start

```

#### Client

```
$ cd client
$ npm install
$ npm run serve

```
